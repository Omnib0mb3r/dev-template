## Feature Workflow

All feature and change work follows the superpowers pipeline. Start here for any new feature, bug, or improvement:

1. **Brainstorm** (`/brainstorming`) — Design the feature collaboratively. Produces a spec in `docs/superpowers/specs/`.
2. **Plan** (`/writing-plans`) — Auto-invoked at end of brainstorm. Produces a task-by-task plan in `docs/superpowers/plans/`.
3. **Execute** (`/subagent-driven-development`) — Agents implement the plan task-by-task with spec + quality review after each task.
4. **Finish** (`/finishing-a-development-branch`) — Merge, create PR, or discard. Auto-invoked at end of execution.

Record outcomes (decisions, what was built, next steps) in Decisions Made and Session Log below.
